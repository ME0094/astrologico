🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. AUTOMATED SETUP (Recommended)
   $ bash setup_ai.sh
   
2. MANUAL SETUP
   $ cp .env.example .env
   $ nano .env                          # Add your API keys
   $ source venv/bin/activate
   $ pip install -r requirements.txt

3. START API SERVER
   $ python3 api_server.py
   
4. ACCESS FEATURES
   • Interactive Docs:     http://localhost:8000/docs
   • CLI Client:          python3 api_client.py status
   • Python API:          from ai_interpreter import AstrologicalInterpreter
   • REST API:            curl http://localhost:8000/api/v1/status

💻 COMMON COMMANDS
═══════════════════════════════════════════════════════════════════════════════

# Generate chart with AI interpretation
python3 api_client.py chart --date "2000-01-15 09:30:00" --lat 40.7128 --lon -74.0060

# Ask astrological question
python3 api_client.py ask "What does Mars in Scorpio mean?" \
  --date "2000-01-15 09:30:00" --lat 40.7128 --lon -74.0060

# Analyze compatibility between two people
python3 api_client.py compatibility \
  --date1 "2000-01-15" --lat1 40.7128 --lon1 -74.0060 \
  --date2 "1995-06-20" --lat2 40.7128 --lon2 -74.0060

# Docker deployment
docker-compose up -d

🔐 CONFIGURATION
═══════════════════════════════════════════════════════════════════════════════

Set these environment variables in .env:

AI_PROVIDER=openai                    # AI provider: "openai" or "anthropic"
OPENAI_API_KEY=sk-...                # Get from: https://platform.openai.com/api-keys
ANTHROPIC_API_KEY=sk-ant-...         # Get from: https://console.anthropic.com
API_HOST=0.0.0.0                     # API listen address
API_PORT=8000                         # API listen port
LOG_LEVEL=info                        # Logging level
