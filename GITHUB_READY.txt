🚀 HOW TO UPLOAD TO GITHUB
═══════════════════════════════════════════════════════════════════════════

THREE OPTIONS AVAILABLE:

1️⃣  QUICKEST: GitHub CLI (ONE COMMAND)
   ─────────────────────────────────
   
   Command:
   $ gh repo create astrologico --public --source=. --remote=origin --push
   
   Requirements:
   - Installed: sudo apt-get install gh
   - Authenticated: gh auth login

2️⃣  SECURE: SSH Key (Recommended for developers)
   ────────────────────────────────────
   
   $ git remote add origin git@github.com:YOUR_USERNAME/astrologico.git
   $ git branch -M main
   $ git push -u origin main
   
   Setup:
   a) Generate SSH key: ssh-keygen -t ed25519
   b) Add to GitHub: https://github.com/settings/keys
   c) Paste: cat ~/.ssh/id_rsa.pub

3️⃣  SIMPLE: HTTPS Token (No SSH setup)
   ────────────────────────────────────
   
   $ git remote add origin https://TOKEN@github.com/YOUR_USERNAME/astrologico.git
   $ git branch -M main
   $ git push -u origin main
   
   Setup:
   a) Create token: https://github.com/settings/tokens/new
   b) Scope: repo (full control)
   c) Expiration: 90 days

📋 STEP-BY-STEP GITHUB UPLOAD
═══════════════════════════════════════════════════════════════════════════

Step 1: Create GitHub Repository
  1. Go to https://github.com/new
  2. Repository name: astrologico
  3. Description: Professional astrological calculator with AI
  4. Choose: Public
  5. DO NOT init with README (we have one)
  6. Click "Create repository"

Step 2: Get Your Credentials Ready
  Option A (Easiest): GitHub CLI
    $ sudo apt-get install gh
    $ gh auth login
    
  Option B: Personal Access Token
    - Go to https://github.com/settings/tokens/new
    - Create token with "repo" scope
    - Copy the token
    
  Option C: SSH Key
    - Check: ls ~/.ssh/id_rsa
    - If not exists: ssh-keygen -t ed25519
    - Add to GitHub: https://github.com/settings/keys

Step 3: Push Your Code
  
  FOR GITHUB CLI:
  $ gh repo create astrologico --public --source=. --remote=origin --push
  
  FOR SSH:
  $ git remote add origin git@github.com:YOUR_USERNAME/astrologico.git
  $ git branch -M main
  $ git push -u origin main
  
  FOR HTTPS TOKEN:
  $ git remote add origin https://YOUR_TOKEN@github.com/YOUR_USERNAME/astrologico.git
  $ git branch -M main
  $ git push -u origin main

Step 4: Verify Upload
  Your repository will be live at:
  https://github.com/YOUR_USERNAME/astrologico
